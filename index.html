Online Headbanger Detection System

Single-file browser demo for basic head tracking on a static GitHub Pages site. It uses MediaPipe Face Landmarker in video mode, tracks one face, draws the face mesh, and reports a neutral-pose-calibrated yaw, pitch, and roll estimate.
