Online Headbanger Detection System

Worker-based static web demo with Face Landmarker, Hand Landmarker, and Pose Landmarker. All heavy inference runs inside a background Web Worker. The main thread handles the HTML UI, overlay rendering, and the small 3D avatar preview.
